最长上升子序列nlogn做法（只输出长度）
用一个栈保存最长子序列，当栈顶元素小于新加入的值时，直接进栈，
当栈顶元素大于新加入的值时，用二分找到第一个大于等于它的数，并替换，
最后栈的大小为最大上升子序列,栈中的序列非最长上升子序列。

如果需要输出最长上升子序列，可以用一个和原数组长度相同的数组，记录原
数组在栈中的位置，然后从右到左递推
